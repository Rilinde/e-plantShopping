# e-plantShopping

This project is an online plant shopping application called **Paradise Nursery**.

## Features
- Landing page with background image and welcome message
- Product listing with multiple plant categories
- Add to Cart functionality
- Redux-based state management
- Shopping cart with quantity updates and total calculation
- Fully responsive UI

## Technologies Used
- React
- Redux Toolkit
- Vite
- CSS
